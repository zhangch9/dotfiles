# Configurations for [`Helix`](https://helix-editor.com/)

## Usage

```bash
# (Optional) backup the existing configuration directory
mv "${XDG_CONFIG_HOME}/helix" "${XDG_CONFIG_HOME}/helix.bak"
ln -s "$(pwd -P)" "${XDG_CONFIG_HOME}"
```
