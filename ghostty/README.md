# Configurations for [`Ghostty`](https://ghostty.org/)

## Usage

```bash
# (Optional) backup the existing configuration directory
mv "${XDG_CONFIG_HOME}/ghostty" "${XDG_CONFIG_HOME}/ghostty.bak"
ln -s "$(pwd -P)" "${XDG_CONFIG_HOME}"
```
