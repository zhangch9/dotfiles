# Configurations for [`tmux`](https://github.com/tmux/tmux)

## Usage

```bash
# (Optional) backup the existing configuration directory
mv "${HOME}/.tmux.conf" "${HOME}/.tmux.conf.bak"
ln -s "$(pwd -P)/tmux.conf" "${HOME}/.tmux.conf"
```
