# The `config` file for [OpenSSH](https://www.openssh.com/) clients

## Usage

```bash
# (Optional) backup the existing configuration file
mv ${HOME}/.ssh/config ${HOME}/.ssh/config.bak
cp config ${HOME}/.ssh/config
```
